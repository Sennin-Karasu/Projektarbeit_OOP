
# 3 Zielsetzung

Ziel der Projektarbeit ist die Entwicklung eines funktionalen Wissensmanagement Prototyps im Rahmen des Moduls Objektorientiertes Programmieren. Die Zielerreichung wird anhand klar definierter und überprüfbarer Kriterien gemessen.

## 3.1 Funktionale Ziele

Die Applikation erfüllt folgende messbare Anforderungen:
- Ein Projekt kann mit Name, Kunde, Projektleiter und Kernanforderungen angelegt werden.
- Ein Projekt wird nach dem Speichern korrekt angezeigt.
- Einer bestehenden Projektstruktur können neue Informationen zugeordnet werden.
- Eine Information kann als Text, Bild URL oder Dokument URL erfasst werden.
- Pro Information können maximal drei Tags gespeichert werden.
- Wird versucht, mehr als drei Tags zu vergeben, wird die Eingabe verhindert oder eine Fehlermeldung angezeigt.
- Kommentare, Ergänzungen und Korrekturen können zu einer Information hinzugefügt werden.
- Ergänzungen und Korrekturen sind visuell klar vom Originaltext unterscheidbar.
- Eine Suche nach Tags innerhalb eines Projektes liefert alle Informationen, bei denen mindestens ein Tag übereinstimmt.
- Alle gespeicherten Daten bleiben nach dem Beenden und erneuten Starten der Applikation erhalten.

## 3.2 Nicht funktionale Ziele

Neben der funktionalen Umsetzung werden folgende qualitative Ziele verfolgt:
- Die Anwendung ist objektorientiert aufgebaut und entspricht dem erstellten Klassendiagramm.
- Die Architektur ist klar strukturiert und in logisch getrennte Verantwortungsbereiche unterteilt.
- Die Benutzeroberfläche ist übersichtlich und ermöglicht eine intuitive Bedienung ohne zusätzliche Anleitung.
- Der Code ist nachvollziehbar, modular aufgebaut und wartbar.
- Die Lösung ist für maximal etwa 100 Einträge ausgelegt und arbeitet innerhalb dieses Rahmens stabil.

## 3.3 Zielerreichung

Die Erfüllung der oben definierten Ziele wird durch einen geplanten und dokumentierten Testplan überprüft. Jeder funktionalen Anforderung wird mindestens ein konkreter Testfall zugeordnet. Die Ergebnisse der Tests werden im Kapitel Testdurchführung dokumentiert und bewertet.
