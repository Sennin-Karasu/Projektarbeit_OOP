# Projektarbeit_OOP

## Wissensmanagement-System 

Dieses Projekt ist ein funktionaler Prototyp eines Wissensmanagement-Systems, das im Rahmen des Moduls Objektorientiertes Programmieren entwickelt wurde.
Die Anwendung ermöglicht die strukturierte Verwaltung von Projekten sowie die Erfassung und Kommentierung projektbezogener Informationen.

## Funktionen
- Projekte anlegen und anzeigen
- Wissenseinträge (Text / URL) erfassen
- Kommentare, Ergänzungen und Korrekturen hinzufügen
- Tag-basierte Suche innerhalb eines Projekts
- Persistente Speicherung der Daten (JSON)

## Voraussetzungen
- Python fähige IDE 
- Python 3.10+
- PySide6 `pip install -r requirements.txt`

## Starten der Anwendung
`python -m src.app`
